# LoreBook

Социальный архив людей в твоей жизни: каждый человек — персонаж, каждое знакомство — сюжетная линия.

Карточки персонажей, главы лора, цитаты, gift memory, RPG-статы, пасхалки и шаринг профиля по коду — без соцсети, но с возможностью показать часть лора другу.

## Структура проекта

```
LoreBook/
├── index.html          # точка входа, подключает шрифты/скрипты/стили
├── style.css            # дизайн-токены (цвета, шрифты) — правь тут
├── app.js               # вся логика и экраны приложения (React)
├── config/
│   └── supabase.js       # ключи и настройки Supabase
├── database/
│   └── schema.sql        # SQL-схема базы данных
├── assets/
│   ├── images/
│   └── icons/
```

## Запуск

Это статический сайт без сборки — открывается прямо в браузере через любой хостинг статики.

1. **База данных**: зайди в свой проект на [supabase.com](https://supabase.com) → SQL Editor → вставь и выполни содержимое `database/schema.sql`
2. **Ключи**: в `config/supabase.js` уже вписаны `url` и `anonKey` твоего проекта — если создаёшь новый проект, замени их (Supabase → Project Settings → Data API)
3. **Деплой**: залей всю папку `LoreBook` (файлы из корня, не саму папку) в репозиторий на GitHub → Settings → Pages → включи для ветки `main`

⚠️ Открывать `index.html` напрямую двойным кликом (`file://`) не будет работать — браузеры блокируют часть скриптов для локальных файлов. Нужен реальный `https://` адрес (GitHub Pages подходит).

## Дизайн

Шрифты и цвета вынесены в переменные в начале `style.css` (`--font-display`, `--font-body`, `--color-accent` и т.д.) — меняешь значения там, не трогая `app.js`.

## Технологии

React 18 + Tailwind (через CDN, без сборки) + Supabase (Auth + Postgres + RLS).
